# 时间转换
```
Description

给定一个以秒为单位的时间t，要求用 “::”的格式来表示这个时间。表示时间，表示分钟，
而表示秒，它们都是整数且没有前导的“0”。例如，若t=0，则应输出是“0:0:0”；
若t=3661，则输出“1:1:1”。


Input
输入只有一行，是一个整数t（0<=t<=86399）。


Output
输出只有一行，是以“::”的格式所表示的时间，不包括引号。


Sample Input 1 

0
Sample Output 1

0:0:0
Sample Input 2 

5436
Sample Output 2

1:30:36

```
